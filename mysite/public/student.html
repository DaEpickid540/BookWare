Groq API Key
ARIA runs on Groq's free tier — fast and private
How to get your free Groq key
  1. Go to console.groq.com and sign up (free)
  2. Click API Keys in the left sidebar
  3. Click Create API Key, name it "BookWare"
  4. Copy the key (starts with gsk_) and paste it above
  5. Hit Save Key — ARIA is ready!

Your key is stored only in your browser. It never touches our servers.
